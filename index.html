キーボードショートカット一覧

キー アクション
/ エンティティ検索モード（vi風）
n / N 検索結果の次/前を選択
Tab 選択図形内のハンドルを順にフォーカス
m + マウス移動 全体移動
r + マウス移動 回転
s + マウス移動 拡大縮小
t + マウス移動 接続点の参照元 t 値のスライド
d + マウス移動 接線ハンドルの距離調整
c 円(Circle)を生成
w 選択した2つの図形間に接続ベジェを生成(Wrap)
x 削除
u 元に戻す (Undo)
Ctrl+r / Meta+r やり直し (Redo)
q 保存してギャラリーに戻る
? (Shift+/) このヘルプの表示/非表示
