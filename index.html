MorphDraw

キー アクション
/ 検索モード（vi風）
n / N 検索結果の次/前を選択
Tab 選択図形内のハンドルを順にフォーカス
a アンカートグル（頂点フォーカス中は追加待機）
Enter 頂点追加（Shape割り込み）の決定
ArrowLeft/Right 接続点（頂点）の選択フォーカス
Escape 頂点フォーカス解除
m / r / s 全体移動 / 回転 / 拡大縮小
t / d t値スライド / 接線距離dの調整
c / w 円の生成 / Wrap（膜）生成
x 削除（頂点フォーカス中は頂点削除）
u / Ctrl+r 元に戻す (Undo) / やり直し (Redo)
q 保存してギャラリーに戻る
Shift+w 太さ編集モードへの移行/解除
Shift+s 輪郭の表示トグル
Shift+f 塗りの表示トグル
? この設定・ヘルプの表示トグル

共通設定項目（拡張予定）

【太さ編集】 t+左右ドラッグ:位置移動 | w+上下ドラッグ:太さ変更 | Shift+t+左右ドラッグ:データ位置変更 | x:データ削除 | Shift+w:終了
【パターン角編集】 t+左右ドラッグ:ターゲット移動（角の自動選択） | Shift+t+左右ドラッグ:選択中の角のパラメータ調整 | Shift+p:終了
